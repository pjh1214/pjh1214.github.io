## 서론

프로그래밍을 하다보면 파일이나 폴더를 삭제해야 할 상황이 자주 생깁니다. 특히, 디렉토리(폴더)를 삭제하려고 할 때 "폴더가 비어있지 않다"라는 문제에 부딪히곤 합니다. 이 문제는 폴더 안에 다른 파일이나 하위 폴더가 있는 경우 발생합니다. 이 문제를 어떻게 해결할 수 있는지 살펴봅니다.

## 리눅스와 윈도우에서 폴더 삭제

리눅스와 윈도우에서는 명령어를 통해 폴더를 삭제할 수 있습니다. 각 운영체제에 따른 명령어는 다음과 같습니다.

### 리눅스

리눅스에서는 `rm -r` 명령어를 사용합니다. `-r` 옵션은 '재귀적(recursive)'으로 폴더와 그 안의 모든 파일을 삭제한다는 의미입니다.

```bash
rm -r 폴더이름
```

### 윈도우

윈도우에서는 `rmdir` 또는 `del` 명령어를 사용할 수 있습니다. `/S` 옵션을 추가하면 하위 폴더와 파일까지 모두 삭제합니다.

```cmd
rmdir /S 폴더이름
```

## Python을 이용한 폴더 삭제

Python에서는 `os` 모듈과 `shutil` 모듈을 이용해 폴더를 삭제할 수 있습니다.

### os 모듈

`os.rmdir()` 함수는 비어 있는 폴더만 삭제 가능합니다.

```python
import os
os.rmdir('폴더이름')
```

### shutil 모듈

`shutil.rmtree()` 함수를 사용하면 폴더와 그 안의 모든 파일과 하위 폴더를 삭제할 수 있습니다.

```python
import shutil
shutil.rmtree('폴더이름')
```

## 마무리

폴더 삭제 작업은 보통 간단하지만, 폴더가 비어있지 않을 경우에는 몇 가지 주의사항이 필요합니다. 운영체제의 명령어나 Python의 라이브러리를 활용하면 이 문제를 쉽게 해결할 수 있습니다. 특히, Python을 사용하면 코드 몇 줄로 간단히 처리할 수 있어 매우 편리합니다.
